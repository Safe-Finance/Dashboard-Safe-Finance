# 📈 Safe Finance - Financial Dashboard

[![Next.js](https://img.shields.io/badge/Next.js-000000?style=for-the-badge&logo=nextdotjs&logoColor=white)](https://nextjs.org/)
[![React 19](https://img.shields.io/badge/React_19-20232A?style=for-the-badge&logo=react&logoColor=61DAFB)](https://react.dev/)
[![Prisma](https://img.shields.io/badge/Prisma-2D3748?style=for-the-badge&logo=prisma&logoColor=white)](https://www.prisma.io/)

A feature-rich financial dashboard built for clarity, speed, and security.

## 🌟 Key Features

- **📊 Interactive Analytics**: Real-time visualization of income and expenses using Recharts.
- **🎯 Goal Tracking**: Set and monitor financial milestones with intuitive progress indicators.
- **🔒 Secure Architecture**: Robust data protection with BcryptJS and JWT authentication.
- **⚡ Modern Performance**: Leveraging React 19 and Next.js 16 (beta) for an ultra-responsive UI.
- **🌗 Theme Support**: Seamless switching between light and dark modes.

## 🛠️ Tech Stack

- **Framework**: [Next.js](https://nextjs.org/)
- **Frontend**: [React 19](https://react.dev/), [Radix UI](https://www.radix-ui.com/), [Tailwind CSS](https://tailwindcss.com/)
- **Animations**: [Framer Motion](https://www.framer.com/motion/)
- **Data Visualization**: [Recharts](https://recharts.org/)
- **ORM**: [Prisma](https://www.prisma.io/)
- **Database Support**: AWS RDS, Neon, Postgres, Redis
